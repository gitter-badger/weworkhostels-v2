### Description
Job board for hostels and backpackers.

### Getting Started
* `npm install`
* `npm run build`
* `npm start`
* Navigate to http://localhost:1337

### Development with nodemon
* `npm install`
* `npm install -g nodemon`
* `npm run build`
* `nodemon server.js`
* Navigate to http://localhost:1337

### Tests
Run React Component unit tests
`npm test`
