### Getting Started
Install node modules.
```
npm install
```

Bundle and transpile ES6.
```
npm run build
```

Start Express server.
```
npm start
```

Open app in browser.
```
open http://localhost:1337
```

### Testing
Run unit tests for JavaScript & React Components.
```
npm test
```

Run [Standard](https://github.com/feross/standard) Style check.
```
npm run lint
```
